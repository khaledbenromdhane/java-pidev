Q: [DEFINITION] : Définissez un terme clé.
R: Java est un langage de programmation de haut niveau, orienté objet et robuste, créé par Sun Microsystems (maintenant Oracle) en 1995. Il se distingue par sa portabilité ("Write Once, Run Anywhere"), fonctionnant sur presque tous les appareils via la Java Virtual Machine (JVM). Largement utilisé, il est populaire pour les applications d'entreprise, web, et mobiles.
